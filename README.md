## Quickstart
1. Clone the repo
    - git clone <repo-url>
2. Install dependencies
    - npm install
    - or
    - yarn
3. Start development server
    - npm run dev
    - or
    - yarn dev
4. Build for production
    - npm run build
    - or
    - yarn build
5. Run tests
    - npm test
    - or
    - yarn test

## Project structure (example)
